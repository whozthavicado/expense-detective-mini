# 🕵️ Expense Detective (Mini Version)

A simple Python tool that helps classify expenses as:

- ✅ Necessary
- ⚠️ Avoidable
- ❌ Impulsive

## 🚀 Purpose
This is a basic prototype of a bigger idea: helping people become more aware of their spending habits using simple logic and psychology.

## 🛠️ Tech
- Python (basic logic: input, if, print)

## ▶️ How to run
```bash
python expense_detector.py
```

## 💡 Future improvements
- AI-based classification
- Web app version
- Personalized financial insights

## 🔥 Vision
Turn this into a full digital product that combines finance + behavioral psychology.
